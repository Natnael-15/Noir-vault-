# Noir Vault Atelier

An avant-garde, high-contrast digital storefront and editorial showcase for a high-end designer techwear clothing label. Displays technical garment studies, detailed materials, structural layouts, and interactive components.

## Features

- **Atelier Materials Explorer:** Interactive investigation of tactile technical fabrics (virgin wool, heavy gabardine, CORDURA).
- **Product Studies:** Deep dives into modular hardware, double-welt pocket geometries, and asymmetric silhouettes.
- **Interactive Audio Engine:** Subtle, high-end user interaction click/swoosh feedback.
- **Responsive Layout:** Adaptive desktop/mobile bento grids and drawer systems.

## Tech Stack

- **Core:** React 19 + TypeScript + Vite
- **Styling:** TailwindCSS v4
- **Animation:** Motion (framer-motion)
- **Icons:** Lucide React

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. Clone the repository and navigate to the project directory:
   ```bash
   git clone <repository-url>
   cd noir-vault
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build the production package:
   ```bash
   npm run build
   ```
